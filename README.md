## Project Overview  ##

This project presents an end-to-end analysis of **Meta Ads (Facebook and Instagram)** using **Power BI**. The dashboard provides a view of campaign performance and helps evaluate how ad formats, audiences, budgets, and time periods influence engagement, conversions, and return on investment (ROI).

The objective is to support marketing teams with clear, data-driven insights that improve ad effectiveness and optimize spending.

### Tools 

| Tool      | Purpose                            |
| --------- | ---------------------------------- |
| Power BI  | Dashboard creation & interactivity |
| DAX       | Custom measures & calculations     |
| Excel/CSV | Data source preparation            |

### Key Metrics Tracked

* Impressions, Clicks, Engagements
* Conversions & Purchases
* CTR, Engagement Rate, Conversion Rate
* ROI and Budget Utilization

### Highlights

* Platform-wise performance comparison (Facebook vs Instagram)
* Ad format analysis (Video, Stories, Image, Carousel)
* Audience insights by age group, gender, and location
* Daily, weekly, and hourly engagement trends
* Calendar-based analysis to identify peak performance periods

### Key Insights

* Campaigns achieved **strong reach and engagement**, indicating effective creatives and targeting
* **Video and Stories ads** consistently outperformed other formats in CTR and engagement
* Engagement rates exceeded industry benchmarks
* Conversion performance lagged behind engagement, revealing an opportunity to improve the post-click funnel

### Audience Analysis

* Female users contributed the highest engagement
* The **18â€“30 age group** showed the strongest response to ads
* Engagement declined significantly beyond age 35

###  Recommendations

* Allocate a higher budget to **Video and Stories ads** to maximize ROI
* Improve landing pages and checkout flows to increase conversion rates
* Align campaign schedules with high-engagement time windows
* Focus targeting strategies on high-performing audience segments

### Conclusion

This dashboard delivers a clear and actionable overview of Meta Ads performance by consolidating complex data into intuitive visual insights. It enables stakeholders to quickly identify strengths, detect inefficiencies, and make confident decisions to improve marketing outcomes and reduce wasted ad spend.
